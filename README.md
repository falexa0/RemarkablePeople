# RemarkablePeople
Este projeto como ideia inicial será uma plataforma de entre-ajuda entre pessoas. Voluntários e outros, poderão se registar e falar com o objetivo de ajudar com artigos de primeira necessidade, outras pessoas. Com esta plataforma vai ser mais fácil de ajudar e ser ajudado.


## Requisitos
- Login
- SignUp
- Perguntar aos utilizadores qual a tarefa a realizar como utilizador. (Utilizador ou Voluntário).
- É possivel criar publicações
- Seria criada uma especie de rede social
